
Инструкция по инсталяции.

Необходимо иметь web server, mysql server, php 5.

На web сервере в корне создадим директорию и поместим туда файлы.

В файле ide/db_params.php прописываем свои реквизиты для подключени к MySQL

Создадим в MySQL новую ДБ из командной строки, phpMyAdmin или другим удобным способом.

В фале db/dump_db.sql находится дамп БД. Восстановим дамп в соданную БД с помощью recover_db.bat или другим удобным способом.

В броузере набираем <DOMEN>/<созданая директория>/index.php
